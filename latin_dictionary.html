Step 2

Open a text editor (such as notepad for Windows or TextEdit for Mac) on your computer


Step 3

Create a new document in the text editor


Step 4

Paste into the text editor (by pressing Ctrl + v for Windows or Command + v for Mac. Alternatively you can right click in the editor and then click the button that says Paste)


Step 5

Save the document as an html file. This is done by pressing Ctrl + s or Command + s or clicking the file button and then clicking Save. The save window should then pop up. Next, set the "Save as type" to "All Files". Then set the File name to "Wheelocks Latin Dictionary.html". Click save or press enter.


Step 6

Now you just need to open the dictionary. Find the file in your computer's File Explorer and double click it. It should then open the dictionary in a web browser. If it does not you will need to right click the file and click "Open with" and then select the web browser of your choice


You now can use the latin dictionary without needing internet access!
